Track a new leader

Signal builds its digest in a scheduled GitHub Action (no API key needed), so leaders are configured in the repo rather than the browser. To add one, edit config/leaders.json and add an entry with the YouTube channel @handle(s) they post on:

{
  "id": "lisa-su",
  "name": "Lisa Su",
  "title": "CEO — AMD",
  "color": "#6e1a3a",
  "aliases": ["Lisa Su"],
  "channels": ["@AMD"]
}

Commit the change — the daily workflow (or a manual Run workflow from the Actions tab) regenerates data/digest.json and this page updates automatically. See the README for the full setup.
